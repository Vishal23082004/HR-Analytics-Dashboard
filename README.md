# 👥 HR Analytics Dashboard

> **Interactive Power BI dashboard for workforce attrition analysis, retention strategy, and HR KPI tracking**

---

## 🎯 Project Overview

This project delivers a comprehensive **HR Analytics solution** for a workforce of **1,470 employees**, built across two interconnected Power BI dashboards:

1. **HR Analytics Dashboard** — Attrition analysis and key turnover drivers
2. **HR Strategic Analytics Dashboard** — Promotion eligibility, retrenchment risk, and satisfaction KPIs

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|---|---|
| Power BI Desktop | Dashboard design & visualization |
| DAX | Automated KPI measures & conditional logic |
| Power Query | Data cleaning & transformation |
| Excel | Data preparation & initial analysis |

---

## 📋 Dashboard 1 — HR Analytics Dashboard

**Period:** November – December 2025

| Metric | Finding |
|---|---|
| Overall Attrition Rate | **19%** across 1,470 employees |
| Highest Risk Segment | **83% attrition** in lowest salary tier |
| Age Group at Risk | **26–35 years** — highest exit concentration |
| Tenure Risk Window | Identified via tenure-education correlation |

### Key Features
- ✅ Department-level attrition drill-down
- ✅ Salary tier vs. attrition heatmap
- ✅ Age group segmentation
- ✅ Tenure & education correlation analysis
- ✅ Targeted retention insights for leadership

---

## 📋 Dashboard 2 — HR Strategic Analytics Dashboard

**Period:** December 2025 – January 2026

### 3-Page Interactive Dashboard covering:

| Page | Content |
|---|---|
| Page 1 | Service Years, Job Level, Job Satisfaction KPIs |
| Page 2 | Promotion eligibility list (DAX automated) |
| Page 3 | Retrenchment risk candidates (DAX flagged) |

### Key Features
- ✅ DAX measures to **auto-flag promotion-due employees**
- ✅ DAX measures to **identify retrenchment candidates**
- ✅ Commute distance vs. attrition & performance analysis
- ✅ Overtime vs. job satisfaction burnout trend analysis
- ✅ Drill-through navigation to individual employee lists
